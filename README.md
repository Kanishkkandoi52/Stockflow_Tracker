# Stockflow Tracker

An interactive investment portfolio management application with a comprehensive onboarding flow and dashboard.

## Features

### Onboarding Flow
- **Preferences Setup**: Configure notification and display preferences
- **Portfolio Categorization**: Define investor type (Conservative/Moderate/Aggressive)
- **Investment Goals**: Set short-term and long-term objectives
- **Experience Level**: Specify years of investing experience
- **Market Interests**: Select preferred market sectors
- **Risk Tolerance**: Define risk comfort level
- **Broker Integration**: Connect with major platforms (E*TRADE, Robinhood, Coinbase, etc.)

### Dashboard
- **Market Highlights**: Personalized pre-market and closing updates
- **Portfolio Diversification**: Visual pie chart showing asset allocation
- **Performance Tracking**: 6-month performance visualization
- **Activity Feed**: Recent transaction history
- **Sector Distribution**: Industry sector breakdown
- **AI Assistant**: Interactive chatbot for dashboard customization

## Technology Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API for charts

## Getting Started

1. Clone the repository:


2. Open `index.html` in your browser
